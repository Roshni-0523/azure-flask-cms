Welcome to Azure CMS

A simple cloud-based Content Management System built with modern cloud technologies.


Technologies Used

  • Flask (Python Backend)
  • Microsoft Azure Web App
  • Azure SQL Database
  • GitHub Continuous Deployment

Features

  • User authentication system
  • Create and manage blog posts
  • Dashboard to view all content
  • Cloud hosted on Azure
  • Integrated with GitHub for deployment
